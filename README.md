# BB Extension Sorter

Органайзер меню расширений SillyTavern. Помогает разложить расширения по папкам, перенести их между колонками и сохранить удобный порядок в настройках.

## Возможности

- **Папки** - создавайте группы расширений прямо в меню настроек.
- **Сворачивание** - папки можно раскрывать и скрывать кликом.
- **Цвета** - у каждой папки есть собственный цвет для быстрой навигации.
- **Drag-and-drop** - отдельное окно сортировки позволяет перетаскивать расширения между левой и правой колонкой или внутрь папок.
- **Сохранение раскладки** - структура сохраняется в настройках SillyTavern и применяется автоматически.

## Как пользоваться

1. Откройте настройки расширений.
2. Нажмите кнопку `Управление сортировкой`.
3. Создайте папки и перетащите расширения в нужные места.
4. Нажмите `Применить`.

## Установка

1. Откройте SillyTavern.
2. Перейдите в `Расширения -> Установить расширение`.
3. Вставьте ссылку на репозиторий:
   `https://github.com/maxkara14/BB-Extension-Sorter`
4. Перезагрузите страницу.
5. Нажмите `Управление сортировкой` над списком расширений.
