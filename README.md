**🗂 BB Extension Sorter**

Расширение для SillyTavern, которое приносит идеальный порядок в ваше меню расширений.

**Что внутри?**

* **🗂 Папки:** Создавайте папки и сортируйте ваши расширения. Они поддерживают сворачивание/разворачивание кликом и не мешают пользоваться вашими расширениями даже внутри папки.
* **🕹 Удобное управление:** Отдельное окно для быстрой сортировки. Перемещайте плагины между левой и правой колонкой или закидывайте их внутрь папок обычным перетаскиванием мыши.
* **🎨 Кастомизация:** Возле каждой папки есть иконка выбора цвета. Раскрашивайте папки по категориям для визуальной навигации.

---

**⚙️ Установка:**

1. Откройте SillyTavern.
2. Перейдите во вкладку *Расширения -> Установить расширение*.
3. Вставьте ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-Extension-Sorter`
4. Перезагрузите страницу (F5).
5. Нажмите кнопку «Управление сортировкой» над списком ваших расширений и создайте свою первую папку.
